# Paul & Greg - World Wide Bar & Cocktails Services

## Overview
A single-page marketing website for Paul & Greg, a bar and cocktail events company. The site is in Hebrew (RTL) with English branding elements.

## Project Architecture
- **Type**: Static HTML website
- **Styling**: Tailwind CSS (CDN), Google Fonts (Playfair Display, Heebo, Great Vibes)
- **Icons**: Material Icons
- **Server**: Python HTTP server (development), static deployment (production)

## Structure
- `public/index.html` - Main website HTML
- `server.py` - Development server (Python, port 5000)

## Key Features
- Full-screen hero with brand identity
- Client logos section (Deloitte, ironSource, Yahoo, MoonActive, Amdocs)
- About section with overlapping image layout
- Photo gallery grid
- Testimonials section
- Instagram feed section
- FAQ accordion
- Footer with contact info, services, social links
- Fixed WhatsApp button and contact sidebar
- Dark mode support
- RTL layout for Hebrew content
